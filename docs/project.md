# 🚀 Internship Project

# Automated Certificate Generation Workflow

---

## 📌 Project Overview

During my internship at **iGradAI Labs Private Limited**, I developed an **Automated Certificate Generation Workflow** using **n8n**. The workflow automates the process of generating certificates and sending them to recipients, reducing manual effort and improving efficiency.

---

## 🎯 Project Objective

The objective of this project was to:

- Automate certificate generation.
- Reduce manual work.
- Improve speed and accuracy.
- Learn workflow automation using n8n.
- Integrate different services within a single workflow.

---

## 🛠️ Technologies Used

- n8n Workflow Automation
- GitHub
- GitHub Pages
- Google Drive
- Google Slides
- Gmail
- MkDocs

---

## 🔄 Workflow

```text
Student Data
      │
      ▼
Generate Certificate
      │
      ▼
Convert to PDF
      │
      ▼
Store in Google Drive
      │
      ▼
Send Email to Student
```

---

## ✨ Features

- ✅ Automatic certificate generation
- ✅ Workflow automation using n8n
- ✅ PDF creation
- ✅ Google Drive integration
- ✅ Automatic email delivery
- ✅ Reduced manual effort
